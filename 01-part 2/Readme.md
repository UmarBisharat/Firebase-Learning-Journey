# 🔥 Firebase Setup Checklist

This is my personal checklist for connecting Firebase to a new Flutter project.

## Steps

### 1. Open my Flutter project.

### 2. Open the browser and go to the **Firebase Console**.

### 3. Create a new Firebase project.

### 4. Ignore **Step 1** in the Firebase setup because it has already been completed. There is no need to do it again.

### 5. Complete **Step 2** by copying and running the two commands provided by Firebase in the Flutter project's terminal.

### 6. Complete **Step 3** by copying the generated initialization code from the Firebase Console and pasting it into the `main.dart` file.

### 7. Add the `firebase_core` package to the `pubspec.yaml` file.

### 8. Run:

```bash
flutter pub get
```

### 9. If required, update the **NDK version** and **SDK version** in the Android project.

### 10. Run the Flutter project.

🎉 That's it! My Flutter project is now connected to Firebase and ready to use.
